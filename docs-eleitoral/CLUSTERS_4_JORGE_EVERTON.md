```
// Campo    -15      -12     -20      0
// Mercado  -5       -20     -8      -5
// Alheio    5        0      -5      -35
```

### Explicacao de cada forca:

**Base auto-atrai forte (-40)**: Policiais, servidores e comissionados de JE ficam juntos — corporativismo + emprego + identidade. E o nucleo duro.

**Campo auto-atrai fraco (-12)**: Jovens, mulheres e evangelicos NAO se conhecem entre si. Sao dispersos. Baixa coesao natural. So se agrupam quando um candidato os PUXA.

**Mercado auto-atrai fraco (-8)**: Vendedores de voto nao formam comunidade — sao individuais. Cada um negocia por si. Pouca coesao.

**Alheio auto-atrai MUITO forte (-35)**: Indigenas (CIR) e base de Sampaio/Joilma sao MUITO coesos. Dificilimos de desagregar.

**Base atrai Campo (-15/-15 mutuo)**: A base de JE atrai naturalmente quem busca seguranca e ordem. Mas a atracao e MODERADA — o Campo nao vem sozinho, precisa de campanha.

**Campo atrai Mercado (-20/-20 mutuo)**: Jovens e mulheres de periferia SÃO vizinhos do mercado do voto. Convivem. Se misturam. A fronteira e porosa. Essa e a forca mais importante para JE: se conquistar o Campo, parte do Mercado vem junto por proximidade.

**Base repele Alheio (+5/+5 mutuo)**: Tensao entre seguranca/policia e indigenas. Nao e guerra, e atrito institucional.

**Campo neutro com Alheio (0/0)**: Jovens nao tem conflito com indigenas nem com base de Sampaio. Sao indiferentes.

**Mercado atrai Alheio levemente (-5/-5)**: No mercado do voto, todo dinheiro e bem-vindo — inclusive de candidatos do territorio alheio. Sao mercenarios.

## CLASSIFICACAO DE CADA ELEITOR

```javascript
function classifyVoter4(mun) {
    const orient = clamp(mun.bol + randn()*0.12, 0, 1);
    const proGar = clamp(mun.gar + randn()*0.10, 0, 1);
    const proDem = clamp(mun.dem + randn()*0.10, 0, 1);
    const antiImig = clamp(mun.imig + randn()*0.10, 0, 1);
    const social = clamp(mun.soc + randn()*0.08, 0, 1);
    const desinfo = clamp(mun.desinfo/5 + randn()*0.08, 0, 1);

    const isBV = mun.n > 50;
    const isIndigMun = mun.dem > 0.55;
    const r = Math.random();

    // Classificacao por afinidade
    let cluster;

    // Territorio Alheio: indigenas organizados + anti-Bolsonaro convictos
    if (isIndigMun && proDem > 0.50 && orient < 0.50) {
        cluster = 3;
    }
    // Base Fiel: seguranca + alta orientacao + anti-imigracao (perfil policial/militar)
    else if (orient > 0.65 && antiImig > 0.50 && r < 0.25) {
        cluster = 0;
    }
    // Mercado: alta vulnerabilidade + baixa fidelidade natural
    else if (social > 0.25 && orient < 0.75 && r < 0.35) {
        cluster = 2;
    }
    // Campo de Disputa: todos os demais (maior grupo)
    else {
        cluster = 1;
    }

    return cluster;
}
```
